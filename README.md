# Intangible Dotfiles

## Programs
* **Editors:** (neo)vim, busybox vi, doom emacs
* **Messaging:** signal, discord, irc, e-mail (if possible plaintext and no pgp)
* **WM/DE:** xmonad, custom dwm, gnome 3
* **Launcher:** custom dmenu, rofi, standard dmenu
* **Shell:** bash, dash, busybox ash
* **Terminal:** alacritty, st, termite
